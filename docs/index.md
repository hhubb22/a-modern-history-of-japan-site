# 日本现代史 / A Modern History of Japan

这个站点把现有 Markdown 书稿整理成适合连续阅读的网页版本，保留章节结构、图片、表格和中英双语内容。

This site presents the manuscript as a readable web edition with preserved chapter structure, images, tables, and bilingual content.

## 进入阅读

- [阅读中文](zh/index.md)
- [Read in English](en/index.md)

## 使用方式

- 左侧导航按编与章节展开，适合连续阅读。
- 右上角搜索可以直接检索章节标题、人名与术语。
- 图片与表格保留在原章节中，便于和 PDF 结构对照。

## 维护说明

- 内容来源于原始书稿仓库，由 `scripts/sync_content.py` 同步生成。
- 若原稿更新，运行 `make sync` 后再重新构建或部署即可。
